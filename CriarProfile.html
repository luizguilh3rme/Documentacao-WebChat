Etapas para acessar Profile


Acessando painel admin

1º Etapa

Ao clicar em admin conforme mostra a seta vermelha, apareceram várias funcionalidades disponibilizadas pelo sistema, entre elas a criação do Profile.



Acessando Profile

2º Etapa

Ao clicar em Profile conforme mostra a seta vermelha, apareceram todos os Profiles ja integrados ao sistema.



Visualizando Profiles

3º Etapa

Apos clicar em Profile conforme a etapa anterior, conseguimos visualizar um campo onde mostra o profile vinculado ao sistema. O usuário poderá clicar no mesmo para APENAS a visualização do Token e as demais observação, não é possível editar.



Adicionar Profiles

4º Etapa

Ao clicar em Adicionar conforme mostra a seta vermelha, apresentará um modal com todos os campos necessários para serem preenchidos e principalmente adicionando o token do sistema externo desejado para realizar a conexão.



Adicionando Profile

5º Etapa

Apos clicar em Adicionar conforme a etapa anterior, conseguimos visualizar os campos para preencher e adicionar o novo Profile.
