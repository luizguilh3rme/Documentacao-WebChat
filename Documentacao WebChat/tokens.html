Etapas para acessar os tokens


Acessando os tokens

1º Etapa

Ao clicar em Tokens conforme mostra a seta vermelha, conseguimos visualizar os tokens vinculados ao sistema.

Observação: Antes de realizar qualquer procedimento de adicionar um novo Token, é neceassário criar um Profile para a vinculação do token. Se ainda não criou o Profile, aperte no botão abaixo para seguir o passo a passo da criação.



Adicionar tokens

2º Etapa

Ao clicar em Adicionar conforme mostra a seta vermelha, abre um modal onde podemos adicionar um novo token ao sistema do sistema externo desejado.



Adicionando tokens

3º Etapa

Apos clicar em adicionar conforme a etapa anterior, conseguimos visualizar um modal com os campos a serem preenchidos, e o mais importante que é selecionar o Profile.



Editar tokens

4º Etapa

Ao clicar em Editar conforme mostra a seta vermelha, apresentará um modal com o token desejado para ser editado. Nesse processo precisará ter bastante atenção para não causar nada inesperado e fazer a desconexão do sistema.



Editando tokens

5º Etapa

Apos clicar em Editar conforme a etapa anterior, conseguimos editar o token ou até mesmo substituií-lo selecionando um novo Profile.



Apagar tokens

6º Etapa

Ao clicar em Apagar conforme mostra a seta vermelha, conseguimos apagar o token, precisamos apenas realizar a confirmação e apos isso o processo será realizado..
