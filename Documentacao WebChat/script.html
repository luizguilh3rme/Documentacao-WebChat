Etapas para acessar os Scripts


Acessando o scripts

1º Etapa

Ao clicar em Scripts conforme mostra a seta vermelha, conseguimos visualizar os scripts (códigos) do sistema.


Observação: Os scripts estão diretamente relacionados com o fluxo de atendimentos, qualquer alteração nos mesmos afetará o fluxo do chat.



Adicionar scripts

2º Etapa

Ao clicar em Adicionar conforme mostra a seta vermelha, abre um modal onde podemos criar um novo codigo referente as funções no fluxo de atendimentos no sistema.



Adicionando scripts

3º Etapa

Apos clicar em adicionar conforme a etapa anterior, conseguimos visualizar um modal com os campos a serem preenchidos, e o principal que é o código.



Editar scripts

4º Etapa

Ao clicar em Editar conforme mostra a seta vermelha, apresentará um modal com o código desejado para se editado. Nesse processo precisará ter bastante atenção para não causar nada inesperado no fluxo de atendimentos.



Editando scripts

5º Etapa

Apos clicar em Editar conforme a etapa anterior, conseguimos editar o codigo conforme as novas funcionalidades para o fluxo de atendimentos.



Apagar scripts

6º Etapa

Ao clicar em Apagar conforme mostra a seta vermelha, conseguimos apagar o script, precisamos apenas realizar a confirmação e apos isso o processo será realizado..
